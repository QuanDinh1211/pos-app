POST /api/inventory/import -> Tạo phiếu nhập hàng
GET /api/inventory/import -> Danh sách phiếu nhập
GET /api/inventory/import/:id -> Chi tiết phiếu nhập
POST /api/inventory/imports/:id/cancel -> Hủy phiếu nhập

POST /api/inventory/export -> Xuất kho thủ công
GET /api/inventory/export -> Danh sách xuất kho
GET /api/inventory/export:id -> Chi tiết xuất kho
POST /api/inventory/export/:id/cancel -> Hủy phiếu xuất kho


POST /api/inventory/adjust -> Điều chỉnh kho thủ công
GET /api/inventory/adjust -> Danh sách Điều chỉnh kho
GET /api/inventory/adjust:id -> Chi tiết Điều chỉnh kho
POST /api/inventory/adjust/:id/cancel -> Hủy phiếu Điều chỉnh kho

GET /api/inventory/check-inventory:id -> Chi tiết Kiểm kho kho
POST /api/inventory/check-inventory/:id/cancel -> Hủy phiếu Kiểm kho kho

GET /api/inventory/logs -> Tất cả log kho
GET /api/inventory/:id/logs -> Log theo inventory
